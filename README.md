# Elixir Tutorial
This is elixir tutorial repository.

---

## 開発環境

### Mac

```
$ brew update
$ brew install elixir
```

### Docker

```
$ docker run -it --rm elixir (bash)
```

---

## 実行

### シェル

```
$ iex
```

### ファイルをコンパイル

```
$ elixir <file-name>
```

---

## 基本型

```
1          # integer
0x1F       # integer
1.0        # float
true       # boolean
:atom      # atom / symbol
"elixir"   # string
[1, 2, 3]  # list
{1, 2, 3}  # tuple
```
